🧠 POA / Dominic System Trainer

Learn Dominic O'Brien's method for memorising numbers — turn digits into People, Objects & Actions.

The big idea

Numbers are hard to remember because they're abstract. The Dominic System fixes this by turning every pair of digits (00–99) into a memorable person, and then into a tiny scene. People doing things stick in memory far better than bare digits.

POA = Person · Object · Action. Each two-digit number gets all three, so a single vivid mental image can lock in six digits at once.

Step 1 — The letter code

Each digit becomes a letter. Memorise these ten — it's the whole foundation:

DigitLetterWhy it's easy to remember
1A1st letter of the alphabet
2B2nd letter
3C3rd letter
4D4th letter
5E5th letter
6S"Six" begins with S
7GG is the 7th letter (A B C D E F G)
8HH is the 8th letter (…F G H)
9N"Nine" begins with N
0O0 looks like the letter O

Tip: 1–5 are just A–E in order. Only 6, 9 (sounds) and 7, 8, 0 (shapes/positions) need a little extra.

Step 2 — From a number to a person

Convert each digit to its letter, read the two letters as initials, then picture a real person with those initials.

27 → 2=B, 7=G → initials B.G. → Bill Gates

15 → 1=A, 5=E → initials A.E. → Albert Einstein

33 → 3=C, 3=C → initials C.C. → Charlie Chaplin

Your list is personal — pick whoever you picture fastest (friends, athletes, characters all work).

Step 3 — Add an Object & an Action (POA)

Give each person a signature object they hold and a signature action they do. For example Einstein (15) → object: chalkboard, action: scribbling equations.

To memorise a longer number, break it into pairs and build one scene using the Person of the first pair, the Object of the second, and the Action of the third:

Number 271533 → pairs 27 · 15 · 33

27 → Bill Gates  (the Person)

15 → Einstein's chalkboard  (the Object)

33 → Chaplin's cane-twirl  (the Action)

→ Picture: "Bill Gates twirling a cane in front of a giant chalkboard." That one image = 271533.

Just starting out? Skip the compression at first — turn each pair into a person and chain them into a little story. Add Objects & Actions once the people come quickly.
